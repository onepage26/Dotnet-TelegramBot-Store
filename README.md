Telegram Coffee Shop Bot Backend

Описания проекта: Бэкенд для Telegram-бота с меню для кофейни и также уведомление приходящее от бота к менеджеру

Что внутри:

Меню: Категории и товары берутся из базы SQLite

Заказы: Бот пересылает детали заказа и юзернейм клиента админу

Стек: .NET 8, EF Core, SQLite.

BPMN-процесс: Обрабатывает заказы онлайн в кофейне через Telegram бота

Описание этапов процесса:

Клиент заходит в Telegram-бот

Выбор категории и товара (автоматизировано через SQLite)

Оформление заказа (нажатие кнопки «Купить»)

Поток А: Бот мгновенно отправляет подтверждение клиенту что заказ принят

Поток Б: Бот мгновенно уведомляет менеджера о новом заказе

Подготовка заказа менеджером (выполняется в реальности)

2. Получение ChatID (для менеджера)

Чтобы заказы приходили тебе (или менеджеру), нужно узнать свой ID

Найди бота @userinfobot и напиши ему любое сообщение. Он пришлет твой Id

3. Настройка конфига

В корне проекта создай файл appsettings.json

Вставь туда этот код и замени данные на свои

JSON
{
  "ConnectionStrings": {
    "DefaultConnection": "Data Source=shop.db"
  },
  "Telegram": {
    "Token": "ВСТАВЬ_HTTP_API",
    "ChatId": "ВСТАВЬ_СЮДА_ID_МЕНЕДЖЕРА"
  }
}
4. Подготовка базы данных
Открой терминал и пропиши: dotnet ef database update

После этого в папке появится файл shop.db. Это твоя база с категориями и товарами

5. Запуск

Нажми кнопку Play/Launch Зайди в свой созданный бот в Telegram и нажми /start.
